Избавляемся от Т модели (*T.mdl)


Довольно простая программа, необходимая для того, чтобы избавиться от Т модели.


Инструкция:
1) Качаем программу.
2) Разархивируем HL1 MDL Compiler & Decompiler.rar
3) Закидываем модель и Тмодель в папку HL1 MDL Compiler & Decompiler.
4) Декомпилируем модель без Т с помощью mdldec12/decompile_all.bat
(вводите название модели, например: zombie.mdl а затем zombieT.mdl и жмете Enter)
5) Удаляем из папки *.mdl и *T.mdl
6) Находим файл с расширением *.qc 
7) Открываем любым текстовым редактором, стираем оттуда строку $externaltextures
8) Закидываем *.qc файл в compiler.bat
(пишем название модели, только вместо .mdl пишем .qc, например: zombie.qc)
9) Ждем, готово, в папке появится моделька без *T.mdl

+инструкция внутри, в формате .txt

Примечания: Главное чтобы путь к папке "HL1 MDL Compiler & Decompiler" был без русских букв.
